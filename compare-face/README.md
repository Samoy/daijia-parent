# 人脸识别服务

这个人脸识别服务只需要会用就可以了

## 环境

- Linux
- MacOS

## 启动

`docker compose up -d`

## 访问

http://localhost:8000/
根据提示一步步进行创建账号、创建项目、上传主题等步骤，然后就可以进行人脸识别了。
